# Dream Garage

A modern, animated website showcasing my dream cars and the vision behind my future lifestyle.

##  Live Concept
This project is more than a car gallery — it's a visual representation of my goals, discipline, and long-term vision.

## Features
- Smooth scrolling
- Scroll-triggered animations
- Responsive car grid layout
- Clean dark luxury UI design

##  Cars Included
- Audi RSQ8 Mansory
- Audi RSQ8 ABT
- Dodge Hellcat
- Toyota Probox (first step in the journey)
- Toyota Land Cruiser

## Built With
- HTML5
- CSS3
- JavaScript

##  Purpose
This project reflects my mindset:
> “This is not a dream. It’s a plan.”

It also serves as part of my journey in improving my web development skills and building real-world projects.
